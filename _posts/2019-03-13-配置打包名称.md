在build.gradle 配置文件中 android  标签下添加以下配置



```
android.applicationVariants.all {
    variant ->
        variant.outputs.all {
            //在这里修改apk文件名
            outputFileName = "oxnightwatch-v" +
                    defaultConfig.versionName + "-" + defaultConfig.versionCode + ".apk"
        }
}
```
